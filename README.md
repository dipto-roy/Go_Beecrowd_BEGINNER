# Go Practice — Beecrowd Beginner Problems

A hands-on repo for **learning Go and mastering its syntax** through solving [Beecrowd](https://www.beecrowd.com.br/) (formerly URI Online Judge) beginner-level problems.

> Goal: Build fluency with Go — variables, types, I/O, math operations, and idiomatic patterns — one problem at a time.

## Solved Problems

| Folder | Problem |
|--------|---------|
| `Simple Product` | Multiply two integers |
| `sum` | Sum of two values |
| `Difference` | Absolute difference of four numbers |
| `Average 1` | Arithmetic average of two numbers |
| `Average 2` | Weighted average |
| `Area` | Area of a geometric shape |
| `Area of a Circle` | Circle area (π × r²) |
| `Salary` | Salary calculation |
| `Salary-with-Bonus` | Salary with bonus |
| `simple-calculate` | Basic arithmetic operations |
| `Sphere` | Sphere volume (4/3 × π × r³) |

## Run Any Solution

```bash
cd <folder>
go run main.go
```

Input is provided via stdin as required by each problem.

## What I Practice Here

- Go syntax and data types (`int`, `float64`, `string`)
- Formatted I/O with `fmt.Scan` and `fmt.Printf`
- Constants and math expressions
- Idiomatic Go structure

## Requirements

- Go 1.18+
